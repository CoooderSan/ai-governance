# 访问控制：黑白名单

## 黑名单约束（硬性禁止）

以下操作严格禁止，违反将立即终止执行：

### 命令黑名单

- `DROP TABLE`（数据库删除）
- `kubectl delete`（K8s资源删除）
- `rm -rf /`（系统文件删除）
- `chmod 777`（权限过度开放）
- `eval()`（代码注入风险）

### 路径黑名单

- `/etc/passwd`（系统敏感文件）
- `/var/lib/kubelet`（K8s核心目录）
- `node_modules/`（依赖目录，禁止直接修改）
- `.git/`（版本控制目录）

### API 黑名单

- 生产环境数据库直连
- 未加密的外部API调用
- 第三方支付接口（需专门审批）

## 白名单约束（允许操作）

### 允许的仓库/模块

- `/src/*`（源代码目录）
- `/test/*`（测试目录）
- `/docs/*`（文档目录）

### 允许的工具

- `read_file`（读取文件）
- `write_file`（写入文件，需确认）
- `edit_file`（编辑文件，需确认）
- `run_command`（执行命令，需确认）
- `grep_search`（搜索代码）
